EOMasters Toolbox

General



The options dialog allows you to configure different functionalities of the EOMasters Toolbox.

Import / Export

Import/Export options
The options of the EOMasters toolbox can be exported and imported to a file. This allows you to use the settings on different computers or restore them after reinstalling SNAP.

Quick Menu

Number of actions displayed
Allows to change the number of actions displayed in the quick menu between 1 and 10.
