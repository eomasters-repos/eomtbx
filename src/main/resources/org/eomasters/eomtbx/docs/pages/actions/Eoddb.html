EO Data DB

On EOMasters there is a database of satellite data and their distributors available. It lists 40 different instruments from more than fifteen sources. It indicates if the data is freely available or if you need to pay for the data or if you can apply for the access. You can filter the database entries, by sensor, by platform and by its thematic application.
