EOMasters Toolbox Basic

  Create System Report

This extension provides the new functionality to create a system report. This allows the user to provide information to the developers about the system. The report can be stored as a text file or copied to the clipboard.
The system report contains information about:

  • Basic System Information
  • Installed Modules
  • Open Products
  • Preferences
  • System Properties
  • Environment Variables
  • System Log (last 100 lines)

The system report can be created via the menu Help -> Create System Report. The following dialog is shown and provides a preview of the report.
