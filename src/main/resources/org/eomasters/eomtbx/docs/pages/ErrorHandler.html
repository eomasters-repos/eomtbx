Error Handler

I hope that you will never see this error handler because this would mean that you have found a bug in the EOMTBX. Please help me to fix this and to further improve the EOMTBX by sending me the error report:

The report is generated, and you can view it in the preview area. You can export and copy the report but you can also send it directly by email or open the EOMasters Forum to report the issue. If you choose to send it by email, you have the option to upload the report to a file sharing service. If so, you need to agree to the Terms of service and the privacy policy of this service at least once.

After agreeing to the Terms of Service and the data policy, the report is automatically uploaded to the file sharing service and a link is added to the email.

If you do not agree you can either copy the text to the clipboard or save it to a text file. Afterwards you can use it in the email.



I'm sorry if an error bothered you but thankful if you send the report to me.
