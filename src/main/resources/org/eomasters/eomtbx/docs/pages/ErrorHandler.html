Error Handler

I hope that you will never see this error handler because this would mean that you have found a bug in the EOMTBX. Please help me to fix this and to further improve the EOMTBX by sending me the error report:

The report is generated, and you can view it in the preview area. You can either upload the file to the file sharing service or copy the text into the clipboard or save it to a text file. If you have uploaded to the file sharing service, a link is automatically added to the mail, in the other cases you need to paste in the content from the clipboard or attach the file to the mail.

I'm sorry if an error bothered you and thankful if you send the report to me.
